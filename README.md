# Python-Based Database Sync System 

This repository is a **demo template** for syncing databases using only Python.  
It is designed to be simple to use, yet can be **extended to production and enterprise-level applications**.  

Everything you need to know is **documented in the code** itself.

---

## Features

- Simple Python-based database synchronization
- Fully configurable via `.env` file for database credentials.
- Extendable for enterprise-level production use
- Clear configuration settings in `config/settings.py`
- Ready for integration with production-grade images if needed.
- User can name docker container of their own choice yet I named mysql and postgres.
- Docker-compose.yaml file setup containers for mysql and postgresql databases you can use them to create testing DBs.
- OR just connect with your own source and target database using prefix SOURCE_DB_ and TARGET_DB_ in .env file.
